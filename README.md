// Simple Encryption of the 26 upper case charactes

A simple java code for encrypting the sequece of characters using very simple way.
This code takes original text as input from the user and produce cipher text using defined value for A to Z.

// How to compile and run the code

(1). Save this code in your computer file as SimpleEncryption.java
(2). Compile the code: javac SimpleEncryption.java
(3). Run the code:     java SimpleEncryption
